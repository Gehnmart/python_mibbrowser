pymibbrowser

Qt-клиент SNMP MIB-браузера для Linux. Опрос устройств, приём и отправка трапов, периодические поллы, наблюдения по порогам, визуализация пропускной способности, сценарии на языке SNMP-команд.

Python 3.10+ PyQt6 pysnmp 7 / pysmi SNMP v1 / v2c MIT License
📖 Открыть руководство ⧉ Репозиторий на GitHub

Что внутри

🌳

Умное дерево MIB

Цепочки одиночных папок сжимаются в один узел; фильтр с поиском по описанию; autoresolve символьных имён.

📊

Live-графики и Port View

pyqtgraph с rate-режимом; 64-битные ifHC-счётчики; плитки интерфейсов с up/down-индикацией и throughput.

🔔

Trap Receiver и Sender

Правила с Run command / Play sound; CIDR accept-list; hex-dump сырого PDU для отладки.

⏱

Polls и Watches

Периодические опросы по списку агентов, предикаты по порогам со stateful-журналом переходов.

💾

Offline-диагностика

Save walk в snmpwalk-формат, Agent Simulator для mock-устройства, Compare для diff'а двух снимков.

🛠

Инструменты сети

Ping, Traceroute, Network Discovery с sysName/sysDescr и bulk-добавлением в список агентов.

📝

Скрипты

iReasoning-стиль: get / walk / set / sleep / if / save. Редактор с reference-вкладкой и примерами.

🔍

Device Snapshot

Один клик — sysInfo + ifTable + HOST-RESOURCES-MIB в удобных панелях.

🧰

Portable build

PyInstaller-tarball ~80 МБ, запускается без системного Python. CI через GitHub Actions на Python 3.10 + 3.12.

Быстрая установка

git clone https://github.com/Gehnmart/python_mibbrowser.git
cd python_mibbrowser
./run.sh           # первый запуск: создаст .venv и скомпилирует MIBs

Для готовой портативной сборки: ./build.sh → dist/pymibbrowser-linux-x86_64/ (≈ 80 МБ запакованного).
